## Installation

```
$ pip install tempus
```

## Usage

#### TIMER
**tempus timer \<duration\>** <br/>
*tempus timer 1m* | *tempus timer 2m15s* | *tempus timer 30s* | *tempus timer 10000s*
```
$ tempus timer 1m
[Enter to start]

|█████████████████████████████████▌                                     | 0:32 |
```
